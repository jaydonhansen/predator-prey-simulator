Serial implementation of Agent-based Predator/Prey simulation

Requirements:
- gcc with C++17 and openmp support
- CMake version supporting C++17

Build and install:
- cd /path/to/predator-prey-simulation
- mkdir build
- cmake ..
- make
To run the application, run "./simulation".

The parameters are hard-coded in Main.cpp, so you can change them if you wish.
